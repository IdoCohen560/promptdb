# PromptDB

## Register
product — the surface is a working tool (a database chat console). It carries brand-tier
craft because it doubles as a portfolio centerpiece, but design serves the interaction first.

## Product purpose
Ask a database questions in plain English; get back the exact SQL, the result table, and a
written answer. A LangGraph agent retrieves the schema, writes a read-only query, self-corrects
on error, executes it, and explains the result. Benchmarked at 69.3% on the Spider dev set.

Two ways in:
1. **Instant demo** — query a bundled sample database (Chinook) with zero setup. Runs on the
   owner's key, capped at 5 free queries per visitor.
2. **Your own database** — run a one-command local connector so the agent queries your live DB
   in place. The data never leaves your machine; only schema and your own results reach the model.
   Bring your own model key (Anthropic, OpenAI, or local Ollama) for unlimited use.

## Users
- **Hiring managers / senior engineers** reviewing this as a portfolio project. Skeptical, fast,
  have seen a hundred "AI SQL" demos. Must feel credible and precise within 30 seconds.
- **Data-curious developers** who want to try natural-language querying on real data without
  handing their credentials to a SaaS.

## Tone
Precise, confident, technical without being cold. It is an instrument, not a toy. It shows its
work (the SQL is the proof) rather than hiding behind a chat bubble. Quietly impressive, never
loud or salesy. The craft is in the restraint and the typography of the data itself.

## Strategic principles
- The SQL and the result table are the heroes. The transformation from English to SQL to answer
  IS the product; design dramatizes that, doesn't decorate around it.
- Trust through transparency: every query shows the generated SQL, the read-only guardrail, and
  the cost. Nothing hidden.
- Earn the monospace. Code and data get a real monospace face; prose does not.
- Fast-feeling. Streaming agent steps (schema → write → execute → answer) make latency legible
  instead of dead.

## Anti-references (what it must NOT look like)
- The dark-mode neon-purple "AI tool" reflex (gradient hero, glowing borders, robot mascot).
- Generic SaaS landing: cream background, rounded cards in a 3-up grid, big-number hero metric.
- ChatGPT-clone: a centered message column with bubbles and nothing else.
- Terminal-cosplay: fake green-on-black CRT that mistakes nostalgia for craft.
